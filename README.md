# Simple-GUI-for-openFPGALoader
Simple GUI for openFPGALoader

Requirements:
PyQt5(Python3) and openFPGALoader are installed.

Only tested TangNano9k FPGA board/Ubuntu Linux.

chmod a+x GUI.py and double click GUI.py at Desktop enviroment.
